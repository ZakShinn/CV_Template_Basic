# CV-Basic

Trang CV cá nhân, dễ chỉnh sửa, deploy miễn phí trên [Vercel](https://vercel.com).

## Hướng dẫn chỉnh sửa

**Toàn bộ:** [`src/HuongDan.md`](src/HuongDan.md) (gồm hướng dẫn thêm mạng xã hội)

| File trong `src/` | Nội dung |
|-------------------|----------|
| `resume-basic.ts` | Thông tin cơ bản CV |
| `resume-advanced.ts` | Kinh nghiệm, học vấn, SEO… |
| `avatar.ts` | Ảnh đại diện |
| `color.ts` | Màu sắc |
| `font.ts` | Font chữ |
| `config.ts` | Bật/tắt hiển thị & toolbar |

## Tech stack

Next.js 15 · React 19 · TypeScript · Tailwind CSS 4
